A BitDev é uma equipe composta pelos alunos do curso de Desenvolvimento de Software Multiplataforma da Fatec - Registro/SP que criou a SowTec, uma empresa fictícia do ramo agropecuário com o objetivo de desenvolver um sistema de visão computacional capaz de auxiliar na separação de toletes de palmito para conserva.
